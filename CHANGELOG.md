# Changelog

## 0.10.0

### Changes

- Add support for Windows CI and testing with WSL (#184)
- Add StackExchange.Redis compatible pub/sub API (#202)
- Add transparent compression support with Zstd and LZ4 backends (#213)
- Add Valkey Search (VSS) command support: FT.CREATE, FT.DROPINDEX, FT._LIST, FT.SEARCH, FT.AGGREGATE, FT.INFO, FT.ALIASADD, FT.ALIASDEL, FT.ALIASUPDATE, FT._ALIASLIST (#225)
